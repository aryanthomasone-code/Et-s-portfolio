PROJECT OMEGA

SENSORY-DRIVEN AUTOMOTIVE INTERFACE

YEAR // 2024

TYPE // PRODUCT DESIGN

PROJECT SUMMARY

Project Omega is a sensory-driven driving interface that replaces traditional screen-based dashboards with intuitive feedback systems using light, haptics, and temperature. By translating vehicle data into physical and visual cues, it reduces cognitive load and enhances driver awareness, creating a more immersive and responsive interaction between driver and vehicle.

01 / OVERVIEW

Project Omega explores a shift away from traditional dashboard systems toward a more immersive, responsive driving experience. It started from a challenge, instead of relying solely on visual information, what if the system integrates other sensory systems like color, motion, and haptic feedback to communicate key driving data in real time. The aim is to reduce cognitive load while enhancing awareness and engagement.

02 / SYSTEM

Lane Assist
Directional lights and steering wheel vibration guide correction when drifting.

Speed Feedback
A color ring shifts green → blue → red with speed; red pulses and alerts when limits are exceeded.

Fuel Efficiency
Slow blue pulses indicate efficiency; faster red pulses signal inefficiency.

Engine Temperature
Heat is felt through the wheel, increasing with temperature and triggering alerts at critical levels.

Tire Pressure
A buzzer intensifies as pressure drops; stable levels remain quiet.

Fuel Capacity
Vibration increases as fuel decreases, escalating at critical levels.

Integrated Feedback System
The new system combines light and haptic feedback with minimal UI, using texture to convey sound and heat which enables intuitive and distraction-free awareness of vehicle status.

03 / DESIGN INTENT

When starting Project Omega, there was no master plan—just a chaotic workspace, a mountain of scrap metal, and an obsessive need to build. Initially intended to be a simple robot frame, the project spiraled into an ongoing engineering challenge.

OVERALL THOUGHTS

Project Omega presents a future-facing approach to automotive interfaces, one that challenges the current day interface which rears its head more towards screens rather than tactile feeling mechanism. One day I hope to continue this project down the line and potentially design a whole car with the design philosophies defined by this project
